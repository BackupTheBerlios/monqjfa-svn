Jfa Tutorial

This was a Wiki previously, but several people asked for a document that can also be printed nicely. Therefore I turned the tutorial contents into an old fashioned document.

The tutorial is available as MonqJfaTutorial.pdf. It makes use of the following example Java files:

  • CompileTest.java is used to check that you have a basic installation of monq.jfa.
  • Example.java serves as an initial example.
  • CountWords.java shows how actions should communicate.
  • ExampleFaAction.java contains a simple FaAction implementation.
  • PatternConflict.java is used to explain how priorities are used to resolves conflicts between patterns.

To get the tutorial as well as the example files, download MonqJfaTutorial-@VERSION@.tar.gz
